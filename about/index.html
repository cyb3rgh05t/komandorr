About Komandorr¶

Project Information¶

Komandorr is a modern, open-source service monitoring dashboard designed to simplify infrastructure monitoring for developers, system administrators, and DevOps teams.

Version¶

Current version: 1.5.6

View Changelog

Features¶

Komandorr provides comprehensive monitoring capabilities:

  • Service Monitoring: Track uptime and performance of HTTP/HTTPS endpoints
  • Plex Integration: Monitor Plex Media Server activities in real-time
  • Traffic Monitoring: Track bandwidth usage per service
  • Multi-Theme UI: Beautiful themes matching popular media server aesthetics
  • Multi-Language: English and German translations
  • RESTful API: Complete API for automation and integration
  • Docker Support: Easy deployment with Docker and Docker Compose

Technology¶

Frontend¶

  • React 18+
  • React Router v6
  • Tailwind CSS
  • react-i18next
  • Lucide Icons
  • Vite

Backend¶

  • FastAPI (Python)
  • Pydantic v2
  • httpx (async HTTP client)
  • uvicorn (ASGI server)
