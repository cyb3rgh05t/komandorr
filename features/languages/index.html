Multi-Language Support¶

Komandorr supports multiple languages for a localized experience.

Supported Languages¶

  • English - Default
  • Deutsch (German)

Changing Language¶

  1. Click your username in top-right
  2. Select Language
  3. Choose your preferred language

All UI elements update instantly!

Contributing Translations¶

See Contributing Guide to add new languages.
