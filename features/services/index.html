Service Monitoring¶

Monitor the health and performance of your services with real-time status checks.

Features¶

  • Real-time health monitoring
  • Response time tracking
  • Status history
  • Error logging
  • Custom check intervals

Service Types¶

  • Website - HTTP/HTTPS endpoints
  • App - Applications
  • Panel - Admin panels
  • Server - Servers
