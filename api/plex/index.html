Plex API¶

API endpoints for Plex integration.

Endpoints¶

Get Configuration¶

GET /api/plex/config

Returns current Plex configuration.

Save Configuration¶

POST /api/plex/config
Content-Type: application/json

{
  "server_url": "http://plex:32400",
  "token": "your-token"
}

Validate Connection¶

POST /api/plex/validate
Content-Type: application/json

{
  "server_url": "http://plex:32400",
  "token": "your-token"
}

Returns validation result.

Get Activities¶

GET /api/plex/activities

Returns current Plex downloads and streams.

Get Downloads (Alias)¶

GET /api/downloads

Alias for /api/plex/activities.

Response Format¶

See API Overview for authentication and error handling.
