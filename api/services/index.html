Services API¶

API endpoints for managing services.

Endpoints¶

List Services¶

GET /api/services

Returns array of all services.

Get Service¶

GET /api/services/{id}

Returns specific service by ID.

Create Service¶

POST /api/services
Content-Type: application/json

{
  "name": "My Service",
  "url": "https://example.com",
  "type": "website",
  "interval": 60
}

Update Service¶

PUT /api/services/{id}
Content-Type: application/json

{
  "name": "Updated Name",
  "interval": 120
}

Delete Service¶

DELETE /api/services/{id}

Manual Check¶

POST /api/services/{id}/check

Triggers immediate health check.

Response Format¶

See API Overview for response format and authentication details.
