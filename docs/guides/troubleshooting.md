# Troubleshooting

Common issues and solutions for Komandorr.

## Connection Issues

**Problem**: Cannot connect to services

**Solution**: Check firewall and network connectivity

## Plex Not Working

**Problem**: VOD streams not appearing

**Solution**: Verify Plex URL and token configuration
