# Contributing

Guide for contributing to Komandorr development.

## Getting Started

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a pull request

## Custom Themes

To create custom themes, refer to the [Themes documentation](../features/themes.md).

## Development Setup

```bash
git clone https://github.com/cyb3rgh05t/komandorr.git
cd komandorr
```

## Code Style

Follow the existing code style and conventions.
