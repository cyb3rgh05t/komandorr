# Features

Discover everything Komandorr has to offer for monitoring your infrastructure and managing your Plex server.

## Core Features

### :material-monitor-dashboard: Service Monitoring

Real-time health monitoring for all your services with customizable check intervals and alerting.

[Learn more →](services.md){ .md-button }

### :material-account-group: Plex Invite Management _(Updated in v2.4.1!)_

Complete Plex invitation system with custom invite codes, multi-badge status indicators, library-specific access control, permission management, OAuth-based redemption, re-invitation support, and comprehensive user account management with automatic cleanup.

[Learn more →](../INVITES.md){ .md-button }

### :material-cached: Enterprise Caching System _(Updated in v3.0.0!)_

High-performance caching layer with 80%+ cache hit rates, expanded Redis coverage (sessions, stats, recent media), reduced Plex API calls by 83% and database queries by 80%. Features background statistics aggregation, automatic cache warming, and optional Redis support for multi-instance deployments.

[Learn more →](caching.md){ .md-button }

### :material-history: Watch History & User Analytics _(New in v2.4.0!)_

Track and analyze Plex viewing history per user with advanced filtering, content type breakdowns, and detailed playback statistics.

[Learn more →](../INVITES.md){ .md-button }

### :material-movie-open-play: VOD Streams

Seamless Plex Media Server integration to monitor and display your video-on-demand content.

[Learn more →](vod-streams.md){ .md-button }

### :material-chart-line: Traffic Monitoring

Track network bandwidth usage with carousel navigation, beautiful visualizations, and historical data analysis.

[Learn more →](traffic.md){ .md-button }

### :material-palette: Themes

Multiple theme options including light, dark, and custom color schemes to match your preference.

[Learn more →](themes.md){ .md-button }
