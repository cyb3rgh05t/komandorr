# Features

Discover everything Komandorr has to offer for monitoring your infrastructure.

## Core Features

### :material-monitor-dashboard: Service Monitoring

Real-time health monitoring for all your services with customizable check intervals and alerting.

[Learn more →](services.md){ .md-button }

### :material-movie-open-play: VOD Streams

Seamless Plex Media Server integration to monitor and display your video-on-demand content.

[Learn more →](vod-streams.md){ .md-button }

### :material-chart-line: Traffic Monitoring

Track network bandwidth usage with beautiful visualizations and historical data analysis.

[Learn more →](traffic.md){ .md-button }

### :material-palette: Themes

Multiple theme options including light, dark, and custom color schemes to match your preference.

[Learn more →](themes.md){ .md-button }
